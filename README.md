 # Synthia - [BETA]
![Synthia logo](logo/logo.png?raw=true)

## install
```
apt update
apt install git
```
```
git clone https://github.com/openspace42/synthia-beta
sudo synthia-beta/install
```

## new project
```
synthia new
```

## run project
synthia always runs a file called main.js so you don't need to specify it
```
synthia [params]
```
